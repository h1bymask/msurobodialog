О ПРОЕКТЕ

В работе предлагается выполнить исследование возможности реализации интеллектуального диалога с роботизированным устройством по заранее заданному сценарию. Заранее срежиссированные диалоги могут применяться для обучения, управления оборудованием и выдаче указаний роботу для выполнения работы по заданному сценарию, и в несложных театрализированных постановках.

ЧТО МЫ ДЕЛАЕМ?

[текст шаблон] Внедрение технологии вспомогательной робототехники для повышения эффективности работы больницы. Робот VITA сможет передвигаться с помощью лифтов и между помещениями в больнице самостоятельно, доставляя лекарства, документацию для медицинских работников.
