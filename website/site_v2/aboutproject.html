Участники

ЦЕЛЬ ПРОЕКТА

Разработать технологию создания интеллектуального диалога человека с роботом по заранее спланированному сценарию.

ЗАДАЧИ ПРОЕКТА

Разработка технологии для построения интеллектуального диалога человека и робота;

Разработка технологии формирования заранее срежиссированного сценария общения человека с роботом;

Постановка нескольких демонстрационных сценариев для воспроизведения в виде видеосъемки, театральной постановки и лекции;

Разработка программных средств и базы данных для визуализации эмоций робота, воспроизведения его речи и создания визуальных и звуковых эффектов;

Отработка технологии усиления эстетических эффектов сценической реализации срежиссированного сценария;

Проведение конкурса постановок между отдельными группами разработчиков.
